# BTCUSDT 永续合约纸面策略说明

> 更新时间：2026-07-13  
> 当前策略：`scalp-tpsl-perp-paper`  
> 当前模式：paper trading，本地模拟成交，不是真实交易所实盘下单。

## 1. 当前实际参数

| 项目 | 当前值 | 说明 |
| --- | --- | --- |
| 市场 | `perpetual` | USDT 本位永续合约 |
| 交易对 | `BTCUSDT` | 当前只保留合约策略测试 |
| 周期 | `1m` | 使用 1 分钟 K 线 |

## 2. 策略目前有哪些功能

当前已经支持：

- 读取本地真实 Binance 合约 1m K 线。
- 读取合约 mark price，并在看板里用标记价估算实时持仓盈亏。
- 允许合约开多。
- 允许合约开空。
- 每次开仓自动写入止盈价和止损价。
- 每轮策略运行前先检查已有持仓是否触发止盈/止损。
- 同一策略同一交易对只允许一个打开的纸面持仓。
- 每轮都保存信号、订单、持仓、账户、回测和绩效快照。
- 看板默认展示合约 `BTCUSDT / 1m`，并过滤掉现货交易噪音。

当前没有支持：

- 没有减仓。
- 没有分批止盈。
- 没有移动止损。
- 没有追踪止盈。
- 没有加仓。
- 没有反手开仓。例如多单止损后，不会在同一轮马上反手开空。
- 没有真实交易所下单。
- 没有真实账户保证金、强平价、资金费率结算。

## 3. 什么时候开多

当没有打开的纸面持仓时，满足以下条件才开多：

1. 最近 3 根 K 线收盘均价 > 最近 9 根 K 线收盘均价。
2. 最新收盘价 > 上一根收盘价。
3. 当前不在冷却期。

开多后：

- 订单方向：`buy`
- 持仓方向：`long`
- 数量：`0.01 BTC`
- 止盈价：开仓价 * `1.0035`
- 止损价：开仓价 * `0.998`

示例：

如果开多价是 `63900`：

- 止盈价 = `63900 * 1.0035 = 64123.65`
- 止损价 = `63900 * 0.998 = 63772.2`

## 4. 什么时候开空

当没有打开的纸面持仓，并且市场是 `perpetual` 时，满足以下条件才开空：

1. 最近 3 根 K 线收盘均价 < 最近 9 根 K 线收盘均价。
2. 最新收盘价 < 上一根收盘价。
3. 当前不在冷却期。

开空后：

- 订单方向：`sell`
- 持仓方向：`short`
- 数量：`0.01 BTC`
- 止盈价：开仓价 * `0.9965`
- 止损价：开仓价 * `1.002`

示例：

如果开空价是 `63748.1`：

- 止盈价 = `63748.1 * 0.9965 = 63524.98165`
- 止损价 = `63748.1 * 1.002 = 63875.5962`

## 5. 什么时候平仓

当前 paper trading 实盘循环只按止盈/止损平仓。

### 多单平仓

多单触发条件：

- 当前价格 >= 止盈价：触发 `take_profit`
- 当前价格 <= 止损价：触发 `stop_loss`

多单盈亏计算：

```text
未实现盈亏 = (当前价格 - 开仓价) * 数量
```

### 空单平仓

空单触发条件：

- 当前价格 <= 止盈价：触发 `take_profit`
- 当前价格 >= 止损价：触发 `stop_loss`

空单盈亏计算：

```text
未实现盈亏 = (开仓价 - 当前价格) * 数量
```

## 6. 回测里的额外平仓规则

回测函数里除了固定止盈止损，还包含趋势反转平仓。

多单：

- 当前价格 >= 止盈价：平仓。
- 当前价格 <= 止损价：平仓。
- 快均线 < 慢均线：下一根价格平仓。

空单：

- 当前价格 <= 止盈价：平仓。
- 当前价格 >= 止损价：平仓。
- 快均线 > 慢均线：下一根价格平仓。

注意：当前 paper trading 实时循环的实际持仓检查主要走固定止盈/止损；趋势反转平仓目前主要体现在回测结果里。

## 7. 什么时候减仓

目前没有减仓逻辑。

当前实现是：

1. 没仓时，满足信号后一次性开 `0.01 BTC`。
2. 有仓时，不再开新仓。
3. 到止盈或止损时，整仓关闭。

也就是说，现在是固定仓位、整进整出。

## 10. 当前策略风险和待完善点

当前策略偏简单，适合先做 paper trading 跑通链路，但不适合直接大资金实盘。

建议后续优先补：

- 减仓：例如盈利到 0.2% 先平一半，剩余仓位移动止损。
- 移动止损：盈利后把止损移动到开仓价附近，减少回撤。
- 反手规则：止损后是否允许反向开仓，需要独立判断，不能盲目反手。
- 波动率过滤：波动过小时不交易，波动过大时降低仓位。
- 成交量过滤：只在成交量放大时开仓。
- 资金费率过滤：资金费率异常时暂停开多或开空。
- 最大连续亏损熔断：连续亏损达到阈值后暂停策略。
- 真正 reduce-only 平仓单：接真实交易所时必须防止平仓单变成反向开仓。
